# 硬件接线图

---

## Board 1 — 从机 (传感器采集节点)

```
                 STM32F103C8T6 最小系统板
                ┌──────────────────────────┐
                │ PA0  ──── DHT11 DATA     │
                │ PA1  ──── 电位器 中间脚   │
                │ PA4  ──── W25Q64 CS      │
                │ PB4  ──── RGB LED R ─1kΩ─┤→ GND
                │ PB5  ──── RGB LED G ─1kΩ─┤→ GND
                │ PB0  ──── RGB LED B ─1kΩ─┤→ GND  (共阴LED:公共脚→GND)
                │ PA9  ──── CH340 RXD      │
                │ PA10 ──── CH340 TXD      │
                │ PA11 ──── TJA1050 RXD    │
                │ PA12 ──── TJA1050 TXD    │
                │ PB1  ──── 按键1 ─┐       │
                │ PA2  ──── 按键2 ─┤→ GND  (按下低电平, 内部上拉)
                │                          │
                │ PB13 ──── W25Q64 SCK     │
                │ PB14 ──── W25Q64 MISO    │
                │ PB15 ──── W25Q64 MOSI    │
                │                          │
                │ SWCLK/SWDIO ─ ST-Link    │
                │ 3.3V ────── 面包板电源轨  │
                │ GND  ────── 面包板 GND轨  │
                └──────────────────────────┘
```

### 面包板详细接线

| 外设 | VCC | GND | 信号 |
|------|-----|-----|------|
| **DHT11** | 3.3V | GND | DATA → PA0 |
| **10k 电位器** | 左→3.3V | 右→GND | 中→PA1 |
| **RGB LED (共阴)** | — | 公共脚→GND | R→PB4(1kΩ), G→PB5(1kΩ), B→PB0(1kΩ) |
| **W25Q64** | 3.3V | GND | CS→PA4, SCK→PB13, MISO→PB14, MOSI→PB15 |
| **TJA1050** | **5V** | GND | RXD→PA11, TXD→PA12, CANH/CANL→总线 |
| **按键1** | — | 一脚→GND | 另一脚→PB1 (内部上拉) |
| **按键2** | — | 一脚→GND | 另一脚→PA2 |
| **CH340 串口** | 3.3V | GND | RXD→PA9, TXD→PA10 |

> 蜂鸣器已移除。PB8 暂无外接模块。

## Board 2 — 主机 (显示监控节点)

```
                 STM32F103C8T6 最小系统板
                ┌──────────────────────────┐
                │ PA11 ──── TJA1050 RXD    │
                │ PA12 ──── TJA1050 TXD    │
                │ PB6  ──── OLED SCL       │
                │ PB7  ──── OLED SDA       │
                │                          │
                │ SWCLK/SWDIO ─ ST-Link    │
                │ 3.3V ────── 面包板电源轨  │
                │ GND  ────── 面包板 GND轨  │
                └──────────────────────────┘
```

| 外设 | VCC | GND | 信号 |
|------|-----|-----|------|
| **SSD1306 OLED** | 3.3V | GND | SCL→PB6, SDA→PB7 |
| **TJA1050** | **5V** | GND | RXD→PA11, TXD→PA12, CANH/CANL→总线 |

> Board2 无串口模块、无 RGB LED、无按键、无蜂鸣器。仅 OLED + CAN。

## CAN 总线连接

```
 Board1 TJA1050                    Board2 TJA1050
     CANH ───────┬─────────────── CANH
                 │
                120Ω (两端各一个)
                 │
     CANL ───────┴─────────────── CANL
```

## 电源轨分配

```
面包板 (两块板共用一个面包板或各用一个)

  上电源轨 (红线):  3.3V  ← STM32板载3.3V引出
  下电源轨 (蓝线):  5V    ← STM32板载5V引出 (或USB供电)
```

| 电压 | 接什么 |
|------|--------|
| 3.3V | DHT11、W25Q64、OLED、电位器、CH340 |
| 5V | TJA1050 ×2 (CAN收发器需要5V) |
| GND | **所有模块的GND必须连通** |
