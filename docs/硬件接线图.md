# 硬件接线图

## Board 1 — 传感器采集节点

```
                STM32F103C8T6 最小系统板
               ┌─────────────────────────┐
               │   PA0 ──── DHT11 (DATA) │
               │   PA1 ──── 10k 电位器   │
               │   PA4 ──── W25Q64 (CS)  │
               │   PA5 ──── RGB LED (R)  │── 220Ω ──┤├── GND
               │   PA6 ──── RGB LED (G)  │── 220Ω ──┤├── GND
               │   PA7 ──── RGB LED (B)  │── 220Ω ──┤├── GND
               │   PA9 ──── CH340 (RXD)  │  串口调试
               │  PA10 ──── CH340 (TXD)  │
               │  PA11 ──── TJA1050(RXD) │
               │  PA12 ──── TJA1050(TXD) │
               │  PB13 ──── W25Q64 (SCK) │
               │  PB14 ──── W25Q64 (MISO)│
               │  PB15 ──── W25Q64 (MOSI)│
               │                         │
               │  ST-Link ── SWCLK/SWDIO │
               │  3.3V ──── VCC (所有外设)│
               │  GND  ──── GND (所有外设)│
               └─────────────────────────┘

               面包板接线:
               - DHT11:  VCC→3.3V, GND→GND, DATA→PA0
               - 电位器: 两端→3.3V/GND, 中间→PA1
               - RGB LED: 共阴, R→PA5(220Ω), G→PA6(220Ω), B→PA7(220Ω)
               - W25Q64: VCC→3.3V, GND→GND, CS→PA4, SCK→PB13,
                          MISO→PB14, MOSI→PB15
               - TJA1050: VCC→5V, GND→GND, TXD→PA12, RXD→PA11,
                          CANH/CANL→总线
```

## Board 2 — 显示监控节点

```
                STM32F103C8T6 最小系统板
               ┌─────────────────────────┐
               │   PA5 ──── RGB LED (R)  │
               │   PA6 ──── RGB LED (G)  │
               │   PA7 ──── RGB LED (B)  │
               │   PA9 ──── CH340 (RXD)  │
               │  PA10 ──── CH340 (TXD)  │
               │  PA11 ──── TJA1050(RXD) │
               │  PA12 ──── TJA1050(TXD) │
               │   PB6 ──── OLED (SCL)   │
               │   PB7 ──── OLED (SDA)   │
               │                         │
               │  ST-Link ── SWCLK/SWDIO │
               │  3.3V ──── VCC (外设)    │
               │  GND  ──── GND (外设)    │
               └─────────────────────────┘

               面包板接线:
               - OLED:   VCC→3.3V, GND→GND, SCL→PB6, SDA→PB7
               - RGB LED: 同 Board1
               - TJA1050: 同 Board1
```

## CAN 总线连接

```
Board1 TJA1050                Board2 TJA1050
    CANH ──────────┬────────── CANH
                   │
                  120Ω
                   │
    CANL ──────────┴────────── CANL
                   │
                  120Ω
                   │
                  GND (仅参考, 不接电源)
```

> ⚠️ CAN 总线的 120Ω 终端电阻必须分别接在两块板子的最远端。
> TJA1050 使用 5V 供电, 但 STM32 的 CAN 引脚是 3.3V 容忍的。
